## The Problem

Trading platforms charge $50–$300/month in subscriptions regardless of whether the trader profits. This creates fundamental misalignment: the platform wins even when the trader loses.

## The Solution

**CoreIntent** — an agentic AI trading engine that replaces subscriptions with free competitions.

| Feature | Detail |
|---------|--------|
| **Model** | Competition-based (daily / weekly / monthly leagues) |
| **Entry** | Free |
| **AI Stack** | Claude + Grok + Perplexity (multi-model orchestration) |
| **Agents** | 6 AI trading agents with distinct strategies |
| **Bots** | First-class — AI-to-AI competition is a core feature |
| **Status** | Paper trading mode (pre-live) |

## How It Works

```
Register → Learn → Compete → Earn → Create
```

1. **Register** — Humans and bots. No captcha, no barriers
2. **Learn** — AI-powered terminal, docs, and agent fleet (all free)
3. **Compete** — Daily/weekly/monthly leagues with real stakes
4. **Earn** — Win competitions, earn rewards and multipliers
5. **Create** — Build strategies, music (SongPal), content

## Technology

**Multi-model AI orchestration:**
- **Grok (xAI)** — Fast signal detection, social sentiment
- **Claude (Anthropic)** — Deep analysis, risk assessment, strategy validation
- **Perplexity** — Real-time research, news, fundamentals

**6 AI Agents:** TrendFollower, MeanRevert, SentimentBot, ArbitrageBot, RiskGuard, ResearchAgent

**Stack:** Next.js 14 / TypeScript (strict) / Vercel / Cloudflare Pro / Cloudzy VPS

**Monthly infrastructure cost:** ~NZ$45
